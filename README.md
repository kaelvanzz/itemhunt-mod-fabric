# itemhunt-mod-fabric
A minigame called "ItemHunt" for minecraft fabric 1.21.4

- required Fabric loader 0.18.4 or newer
- required Fabric API 0.119.4 or newer
you can download the fabric api mod from here: https://www.curseforge.com/minecraft/mc-mods/fabric-api/download/6863303

## Commands
- `/itemhunt config`: shows current gamemode, rounds and roundtime.
- `/itemhunt start`: starts the game
- `/itemhunt end`: force end the game 
- `/itemhunt skip`: force skip the current round
- `/itemhunt rounds <rounds>`: to configure how many rounds (0 on default, unlimited rounds)
- `/itemhunt roundtime <seconds>`: to configure the roundtime (0 on default, unlimited time)
- `/itemhunt mode <colors, name_english, name_chinese>`: set gamemode to **color**, **English first letter** or **Pinyin first letter**.
